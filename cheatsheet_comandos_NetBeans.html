⚙️ Plantilla Personalizada para Archivos Java

Para modificar la plantilla predeterminada de archivos Java en NetBeans:

  1. Abre NetBeans
  2. Ve a Tools (Herramientas) → Templates (Plantillas)
  3. Busca la carpeta Java
  4. Selecciona Java Main Class y haz clic en Open In Editor
Consejo: Personalizar la plantilla te ahorra tiempo al crear nuevos archivos, ya que incluirá automáticamente imports comunes, comentarios Javadoc y configuraciones específicas.

Ejemplo de plantilla personalizada:

// Imports y bibliotecas
import java.util.*;
import java.awt.*;
import java.awt.event.*;
import java.io.*;

/**
* @author ${user}
* @version 1.0
*/
public class ${name} {
  // Inicio del método main
  public static void main(String[] args) {
    // Forzar codificación UTF-8 en la consola
    System.setOut(new java.io.PrintStream(System.out, true, "UTF-8"));
    // Tu código aquí
  }
}

🔠 Forzar Codificación UTF-8 en la Consola

Para habilitar correctamente los caracteres especiales (tildes, ñ, etc.) en la consola de NetBeans:

// Forzar la codificación UTF-8 en la consola
System.setOut(new java.io.PrintStream(System.out, true, "UTF-8"));
Nota: Esta línea debe colocarse al inicio del método main para asegurar que toda la salida a consola use UTF-8.
